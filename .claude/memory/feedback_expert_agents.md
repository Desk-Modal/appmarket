---
name: Expert Agent Profiles for Reviews
description: Use world-leading expert agent profiles for each specific review domain. Commit progress at stable evolution points.
type: feedback
---

When running reviews or audits, each agent should be profiled as the world's leading expert in that specific domain. Generic "audit agent" is insufficient.

**Why:** The user wants the highest quality reviews from specialized perspectives, not generic passes. Also wants git commits at stable evolution points, not just at the end.

**How to apply:**
- E2E testing agent → profile as senior QA automation engineer
- App architecture audit → profile as senior FDC3/trading platform architect
- Package quality audit → profile as senior TypeScript library author / performance engineer
- Security audit → profile as application security specialist
- Commit to git after each meaningful evolution (completed wave, batch of fixes) — not if the codebase isn't in a clean state
